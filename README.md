# Titanic Dataset - Exploratory Data Analysis

## Overview
Exploratory data analysis of the Titanic dataset using Python.
The goal is to understand what factors influenced passenger survival.

## Dataset
- 891 passengers, 12 features
- Source: [Kaggle Titanic Competition](https://www.kaggle.com/competitions/titanic)

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
1. **Gender**: Women survived at 74.2% vs men at 18.9%
2. **Class**: 1st class survival 63% vs 3rd class 24.2%
3. **Age**: Most passengers were 20-30 years old
4. **Overall survival rate**: 38.4%

## Missing Values
- Age: 177 missing values
- Cabin: 687 missing values
